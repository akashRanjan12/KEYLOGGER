# Educational Keylogger Project (github)

## Overview

This project demonstrates keylogging concepts for educational purposes only. It shows how keyboard hooks work in Windows systems while emphasizing ethical considerations.

## Features

- Real-time keyboard monitoring
- Window activity tracking
- Ethical consent mechanism
- Secure logging
- Educational documentation

## Ethical Considerations

⚠️ **IMPORTANT**: This software is for EDUCATIONAL PURPOSES ONLY. Unauthorized use is illegal and unethical.

## Build Instructions

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
